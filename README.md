This repository contains the code, structured extraction matrices, and visualisations developed as part of the systematic review:

## Overview

This repository contains the code and structured matrices used to analyse and visualise data quality dimensions, data quality assessment methods, issue themes, and affected data elements reported in healthcare claims and administrative billing data.

The repository supports the generation of three main visualisations:

1. Data quality dimensions assessed across the included studies.
2. Data quality assessment methods applied in each study.
3. Relationships between data quality issue themes and affected claims-data elements.

---

## Data Statement

This repository contains data extracted from published scientific studies and code used to generate systematic-review figures.

It does not contain patient-level, personally identifiable, confidential, or sensitive healthcare data.
